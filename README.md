# 🌍 World-Go

A modern React application that displays country information including flag, name, and population using REST API data.

🔗 **Live Demo:** https://storied-dolphin-bb3aec.netlify.app/  
🔗 **GitHub Repository:** https://github.com/MohsinAlam07/World-Go  

---

## 📌 About The Project

**World-Go** is a simple yet dynamic React project that fetches country data from a public API and displays it in a clean UI layout.

The application demonstrates:

- Component-based architecture
- Props usage
- API data rendering
- Basic CSS styling
- Modern React development with Vite

---

## 🚀 Features

- 🌎 Display all countries dynamically  
- 🏳 Show country flags  
- 👥 Show population  
- ⚛ Built with functional components  
- 🎨 Clean and responsive design  
- 🌐 Deployed on Netlify  

---

## 🛠 Tech Stack

- **React.js**
- **JavaScript (ES6+)**
- **CSS3**
- **Vite**
- **REST Countries API**
- **Netlify (Deployment)**
